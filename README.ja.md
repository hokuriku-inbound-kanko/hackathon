# hackathon

「Hack! Hokuriku　北陸観光をハックせよ！ハッカソン - データで拓く未来の観光」イベントページ（2024年12月1日開催）。

## 概要

北陸3県（富山・石川・福井）のインバウンド観光に関するアイデアをオープンデータを活用して創出する1日ハッカソン。

参加者は以下のデータを活用して分析を行いました：
- Wi-Fi MACアドレスセンシングデータ
- QRコードアンケート回答データ
- AIカメラ分析データ
- 地域経済分析システム（RESAS）

## テーマ

インバウンド観光という切り口からデータを活用して北陸3県の経済、人が活性化するアプリ作り

## イベント詳細

- **開催日**: 2024年12月1日（日）10時〜17時半（予定）
- **主催**: 北陸インバウンド観光DX・データコンソーシアム
