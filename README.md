Event page for "Hack! Hokuriku — Hack Hokuriku Tourism!" (Hokuriku Inbound Tourism DX Hackathon, December 1, 2024).

## Overview

A one-day hackathon using open data to develop tourism innovation ideas for the Hokuriku region (Toyama, Ishikawa, and Fukui prefectures).

Participants analyzed inbound tourism patterns using:
- Wi-Fi MAC address sensing data
- QR code survey responses
- AI camera analysis data
- Regional Economic Analysis System (RESAS)

## Theme

Build apps that activate the economy and people of the three Hokuriku prefectures through inbound tourism data.

## Event Details

- **Date**: December 1, 2024 (10:00–17:30 JST)
- **Host**: Hokuriku Inbound Tourism DX Data Consortium
